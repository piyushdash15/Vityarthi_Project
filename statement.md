3. Scope of the Project
The scope of this project includes:

1) Designing a GUI-based password evaluation tool
2) Implementing password scoring using length, character diversity, entropy, and pattern detection
3) Real-time analysis using Tkinter event handling
4)Providing clear visual feedback through a strength meter
5) Displaying actionable suggestions for improving weak passwords
6) Ensuring the system is lightweight, offline, and easy to use

Not included in scope (but possible for future updates):

2) Database integration
3) Multi-user management
4) Mobile/Android version
